# 🎉 베이비 인플루언서 팬사인회 랜딩 페이지

만 22개월 아기 인플루언서의 첫 팬사인회 참가자 모집을 위한 랜딩 페이지입니다.

## 📋 프로젝트 개요

귀여운 아기 인플루언서와 함께하는 특별한 팬사인회 이벤트를 위한 참가자 모집 웹사이트입니다. 
선착순 100명 한정으로 모집하며, 참가자 전원에게 타요 출동센터 플레이 세트를 경품으로 증정합니다.

## ✨ 주요 기능

### 1. 히어로 섹션
- 눈에 띄는 대형 배너 with 그라디언트 배경
- 실시간 카운트다운 타이머 (이벤트 날짜까지)
- 선착순 모집 인원 실시간 표시
- 애니메이션 효과가 적용된 CTA 버튼

### 2. 아기 소개 갤러리
- 3개의 귀여운 아기 사진 (사용자 제공 이미지)
- 호버 효과와 캡션이 있는 갤러리 레이아웃
- 반응형 그리드 디자인

### 3. 경품 정보
- 타요 출동센터 플레이 세트 상세 정보
- 제품 이미지 및 특징 소개
- 참가자 전원 증정 뱃지

### 4. 이벤트 상세 정보
- 일시, 장소, 모집 인원, 참가비 정보
- 4개의 정보 카드로 구성
- 아이콘과 함께 직관적인 정보 제공

### 5. 신청 폼
- **보호자 정보**: 성함
- **아이 정보**: 이름, 나이 (선택 옵션)
- **참석 정보**: 참석 인원 (선택 옵션)
- **연락처**: 전화번호 (자동 포맷팅), 이메일 (선택)
- **응원 메시지**: 텍스트 에리어
- **개인정보 동의**: 필수 체크박스
- **실시간 유효성 검사**
- **폼 제출 후 성공/에러 메시지 표시**

### 6. 데이터 저장
- RESTful Table API를 통한 신청 정보 저장
- 자동으로 생성되는 시스템 필드 (id, created_at, updated_at)

## 🗄️ 데이터베이스 스키마

### Table: `fan_meeting_applications`

| 필드명 | 타입 | 설명 |
|--------|------|------|
| id | text | 고유 식별자 (자동 생성) |
| parentName | text | 보호자 성함 |
| childName | text | 아이 이름 |
| childAge | text | 아이 나이 (0-12개월, 13-24개월, 25-36개월, 37개월 이상) |
| attendees | text | 참석 인원 (1명, 2명, 3명) |
| phone | text | 연락처 (010-1234-5678 형식) |
| email | text | 이메일 (선택) |
| message | rich_text | 응원 메시지 |
| submittedAt | datetime | 신청 일시 |

## 🔗 API 엔드포인트

### 신청서 제출
- **Endpoint**: `POST /tables/fan_meeting_applications`
- **Method**: POST
- **Content-Type**: application/json
- **Body 예시**:
```json
{
  "parentName": "홍길동",
  "childName": "홍아기",
  "childAge": "13-24개월",
  "attendees": "2명",
  "phone": "010-1234-5678",
  "email": "example@email.com",
  "message": "우리 아기를 응원합니다!",
  "submittedAt": "2026-01-29T12:00:00Z"
}
```

### 신청 현황 조회
- **Endpoint**: `GET /tables/fan_meeting_applications?limit=1`
- **Method**: GET
- **Response**: 총 신청 건수 및 데이터

## 🚀 배포 방법

1. Publish 탭으로 이동
2. "Publish" 버튼 클릭
3. 자동으로 생성된 URL로 접속 가능

## 📝 사용 방법

1. 페이지 접속 시 카운트다운 타이머가 자동으로 시작됩니다
2. 아기 인플루언서 소개 및 경품 정보를 확인합니다
3. "지금 신청하기" 버튼을 클릭하여 신청 폼으로 이동합니다
4. 필수 정보를 입력하고 개인정보 동의 체크박스를 선택합니다
5. "신청 완료하기" 버튼을 클릭하여 제출합니다
6. 성공 메시지 확인 후 등록된 연락처로 안내를 받습니다
